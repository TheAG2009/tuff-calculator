# VibeCalc - iOS Calculator with Smart Quotes

## ✨ Features
- 🧮 Full iOS-style calculator
- 🎯 Different quotes for different operations:
  - **+** → Motivational quotes
  - **-** → Pick-up lines
  - **÷** → Funny jokes
  - **×** → Action tasks
  - **%** → Deep/meme questions
- 📱 iPhone bezel design
- ⌨️ Keyboard support (Enter = calculate)
- 🎬 Smooth animations (fade + slide)
- 🔄 No duplicate quotes in a row
- 💾 Separate quotes file for easy editing
